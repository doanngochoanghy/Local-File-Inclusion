BÀI 1:  Lỗi local file inclusion

 

Path

Làm 3 đề CTF thể hiện các lỗi web cơ bản

Phụ trách

Khanhpv6

Thời gian

3 ngày

 

Yêu cầu:

-         Xây dựng trang web001 có chức năng upload ảnh gif (chức năng trang web tự chọn ví dụ: ảnh avatar, ảnh sản phẩm, album,..). Trang web001 phải có chức năng đắng kí, đăng nhập người dùng. Sau khi đăng nhập, người dùng mới được sử dụng chức năng upload. Giao diện trang web tùy chọn. (4d)

-         Trang web sử dụng hàm include để tạo các trang con, bị lỗi local file inclusion cho phép đọc được mã nguồn trang web thông qua php wrapper. (1d)

-         Trang web001 sử dụng hàm imagecreatefromgif để check ảnh người dùng upload (khi upload có check phần mở rộng, check tên file và dùng hàm imagecreatefromgif để check đúng định dạng ảnh gif) nhưng hàm imagecreatefromgif có thể bypass để chứa code php (xem link tham khảo bên dưới). (1d)

-         Viết mã khai thác lỗi sử dụng python tự động upshell sau đó đọc và in ra nội dùng file /etc/passwd. (2d)

-         Tìm hiều và bổ sung các cách để chống các user chơi xấu: ví dụ user A sau khi upload được webshell liên tục gửi request để xóa file của người khác; user A viết script tự tạo tài khoàn liên tục dẫn tới đầy database, chậm server; (2d)
